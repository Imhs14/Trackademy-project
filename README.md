## ğŸ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [Git](https://git-scm.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Imhs14/Trackademy-project.git](https://github.com/Imhs14/Trackademy-project.git)
   cd Trackademy-project

```

2. **Install dependencies:**
```bash
npm install
# or
yarn install

```


3. **Set up environment variables:**
Create a `.env` file in the root directory and add your configuration (API keys, DB URIs, etc.):
```env
PORT=5000
DATABASE_URL=your_database_url
SECRET_KEY=your_secret_key

```


4. **Run the application:**
```bash
npm start
# or for development mode
npm run dev

```
